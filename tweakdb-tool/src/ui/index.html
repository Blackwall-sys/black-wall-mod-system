Visão geral

Abrir record

Edite os valores que quiser. Só os campos alterados entram no mod. Use $Nome.Do.Record para referências (TweakDBID) e [a, b, c] para listas.

Tweak declarativo (formato TweakXL .yaml)

Suporta $base (herança), $type (criar do zero), arrays com !append/!prepend/!remove e structs {x,y,z}.

Instalar no jogo

O Aplicar (abas Editar/YAML) gera o arquivo modificado. Instalar copia ele pro lugar que o jogo lê, guardando o original. Desfazer volta ao vanilla. Sempre valide no jogo depois de instalar.
